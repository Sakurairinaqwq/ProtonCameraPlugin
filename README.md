### **三大核心：生灵的视角**

这插件将视角分为三种生存状态，每一种都试图捕捉该“生灵”独特的**情绪**与**重量**。

#### **一、 飞行模式 (Aerial Mode)**

* **意图：** 捕捉自由与速度带来的狂热。为在三维空间中放纵的物体（如战机、无人机）提供视角。
* **特性：**
    * **迟滞的跟随：** 镜头并非机械地钉死在机身后，而是带着一丝**惯性**和**追随欲**，避免那份冰冷的机械感。
    * **身躯的倾斜 (Banking):** 随飞行器转弯时，镜头也带着一种近乎生理性的微微侧倾，让你感觉自己正与铁鸟共呼吸。
    * **距离的敬畏：** 速度越快，视野便越开阔，镜头懂得对极致的速度保持一种必要的**敬畏距离**。

#### **二、 载具模式 (Vehicular Mode)**

* **意图：** 感受地面摩擦与征服的**蛮横**。为地面上那群不肯安分的机器提供视角。
* **特性：**
    * **挣脱与追逐：** 镜头在车辆急转时，会表现出一种短暂的“挣扎”，随后再跟上，像一个不甘落后的随从。
    * **路面的震颤 (Speed Shake):** 根据速度与地表粗粝程度，传递一种细微的、属于金属与沥青摩擦的**震动感**。
    * **低伏的凝视：** 镜头常常压低，带着一种伏击者的姿态，强调速度的压迫性。

#### **三、 人物模式 (Character Mode)**

* **意图：** 回归人类，感受那份**脆弱与坚韧**。为玩家扮演的角色服务。
* **特性：**
    * **重量感的锚定：** 镜头坚定地落在角色的肩后，带着一份不可撼动的**重量感**。
    * **专注的平移：** 当角色集中精神瞄准时，镜头带着一种平稳而坚决的移动，是心无旁骛的体现。
    * **环境的妥协：** 镜头知道如何面对障碍，它懂得在被墙壁遮挡时，迅速而安静地妥协、贴近，不打扰操纵者的思绪。
